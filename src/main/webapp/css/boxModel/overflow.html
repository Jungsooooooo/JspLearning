overflow 속성

컨텐츠가 요소의 영역을 벗어나는 경우의 처리 방법을 설정합니다.

overflow는 컨텐츠가 요소의 영역을 벗어나는 경우 처리 방법을 설정하는 속성입니다. overflow 속성을 방향 별로 지정하기 위한 상세 속성으로는 overflow-x, overflow-y를 사용할 수가 있습니다.
