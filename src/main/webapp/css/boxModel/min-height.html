min-height 속성

1. min-height는 요소의 최소 높이를 설정하는 속성입니다. 컨텐츠의 높이로 계산된 값이 min-height 속성의 값보다 작을 경우에는 요소의 크기는 min-height 값을 표시되며 컨텐츠의 높이로 계산된 값이 min-height 속성의 값보다 클 경우에는 컨텐츠의 높이가 반영되어 높이가 결정됩니다.

2. min-width는 요소의 최소 폭을 지정하는 설정입니다.

3. min-height는 요소의 최소 높이를 설정하는 속성입니다. 컨텐츠의 높이로 계산된 값이 min-height 속성의 값보다 작을 경우에는 요소의 크기는 min-height 값을 표시되며 컨텐츠의 높이로 계산된 값이 min-height 속성의 값보다 클 경우에는 컨텐츠의 높이가 반영되어 높이가 결정됩니다.
